# Personal Portfolio Website

Cream-white and pink themed multi-page site built with HTML/CSS/JS.

Pages:
- Home (`index.html`): headshot, two hobbies, welcome video, intro
- Hobbies (`hobbies.html`): detailed hobbies, 3+ images, stories
- Discover (`discover.html`): Why UMD & Duluth, 30s marketing video, highlights
- Resume (`resume.html`): web-formatted resume, PDF download button
- Career (`career.html`): two companies, contacts, alignment and qualifications
- Game (`game.html`): simple JavaScript game (Catch the Dot)

Customize:
- Replace `media/REM_1491-1.jpg` with your headshot (same name or update paths)
- Add your resume PDF to `media/resume.pdf` to enable download
- Edit text content directly in each HTML file

Run locally:
1. Open `index.html` in a browser, or
2. Serve folder (for best experience):
   - macOS: `python3 -m http.server 5173`
   - then open `http://localhost:5173`

Deploy:
- Use GitHub Pages, Netlify, or Vercel. Deploy the entire folder.
